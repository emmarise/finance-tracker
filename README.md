# Finance Tracker

Track your expenses with natural language. Type something like "Coffee $5, Lunch $12" or paste a multi-day expense log and the app parses it into structured transactions.

**Live:** https://finance-tracker-three-dusky.vercel.app/

## Tech Stack

- **Framework:** Next.js 16 (App Router)
- **Styling:** Tailwind CSS 4 + shadcn/ui (base-nova)
- **Database & Auth:** Supabase (PostgreSQL + Google OAuth)
- **LLM:** Groq (Llama 3.3 70B) for natural language parsing
- **Charts:** Recharts
- **Deployment:** Vercel
